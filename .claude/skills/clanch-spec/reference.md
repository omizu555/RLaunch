# CLaunch 4.22 機能別詳細仕様

出典: 公式ヘルプ CLaunch_ja.chm（4.22同梱）、公式サイト claunch.html / claunch_tips.html /
claunch_newfeatures.html / claunch_skin.html、gigafree.net 等の解説記事。2026-07 調査。

## 1. 呼び出し方法（表示手段）

### 1.1 クリックファンクション
- 任意のマウスボタン（左/右/ホイール/サイド1/サイド2、組み合わせ可）の
  **シングル/ダブル/トリプルクリック、ホイール回転、長押し**に「CLaunch表示」を割り当て。
- 有効範囲: どこでも / 画面の四隅（左上・右上・左下・右下）/ 画面の四辺 / タイトルバー /
  デスクトップ / フォルダーの余白。**デフォルトは「デスクトップのダブルクリック」**。
- 補助オプション: シングルクリック遅延エミュレート（500ms）、ダブル/トリプルのタイムアウト待ち
  （同一ボタンのシングル/ダブル/トリプルに別機能を共存）、カーソル移動で長押しキャンセル、
  低レベルマウスフック使用（ゲーム対策）、長押し認識時間。

### 1.2 エッジファンクション
- カーソルを画面端（四隅+四辺の8箇所）へ移動で表示。
  **デフォルトは「マウスボタンを押しながら画面右下コーナーへ移動」**（チェックを外すと移動のみで表示）。
- エッジ認識範囲を箇所ごとに拡張可（コーナー縦横サイズ、辺の無効領域幅）。
- 表示遅延・非表示化遅延を ms 単位で設定。

### 1.3 サークルファンクション
- マウスで円を描いて表示。時計回り/反時計回り個別割り当て。回転数・最大/最小半径調整。
  左/右/ホイールボタン押下中の無効化オプション。

### 1.4 ホットキー
- 任意のキー + **同一キーの指定回数連打**を条件化できる（例: Ctrl 2回押し）。
- Windowsキー単独はキーリリース時発動で Win+文字キーと共存可能。

### 1.5 その他
- トレイアイコンクリック。多重起動で強制表示（制限モード/フック解除中でも効く）。
- コマンドラインオプション: /n 非常駐モード（非表示操作で終了）、/l ロック状態で表示、
  /m カーソル位置表示、/s X:Y 座標指定、/p ページ指定、/b 横:縦ボタン数、/d 起動遅延、
  /a 自動非表示有効、/nh フック解除起動、/r アイコンキャッシュ破棄。
- **表示操作ごとに初期表示ページを個別指定可**（「前回のページ」も選択可）。
- 表示位置は操作カテゴリ別に: カーソルの周囲（カーソル中心）/ カーソルの右下 / 前回位置 /
  画面中央 / 指定座標。
- **待機表示**: 常時指定位置に表示し、表示操作で対応位置へ移動+アクティブ化、
  用が済むと待機位置へ戻る。

### 1.6 非表示化
- **カーソルがウィンドウ外へ出ると消える**。ON/OFF は操作カテゴリ（クリック/エッジ/サークル/
  キーボード）ごとに設定できるが、非表示化までの遅延時間は「エッジ」タブの共通設定1つが
  全カテゴリに適用される。
- デスクトップ/他アプリのクリック、Esc キーでも消える。
- ウィンドウロック（ピン）中は消えない。

## 2. アイテムのプロパティ（Ctrl+P / Alt+右クリック）

| 項目 | 仕様 |
|---|---|
| 名前 | ボタン/メニュー表示文字列 |
| ファイル名 | パスまたはURL。相対パス（CLaunch.exe基準）・環境変数可。mailto: 可 |
| パラメータ | 引数。`<clipboard>` タグでクリップボード文字列を埋め込み（クリップボード検索ボタン等） |
| 作業フォルダー | カレントディレクトリ |
| メニューオプション | サブメニュー/フォルダー用: /w列幅 /r行数 /c列数 /t /is /il /pX:Y、展開用 /max /in /h /f /s /i、ワイルドカードフィルタ |
| 実行時の大きさ | 通常/最小化/最大化 |
| 優先度 | プロセス優先度（リアルタイム〜低） |
| 特権レベル | 標準ユーザー/管理者として実行 |
| ツールチップ | カーソル静止時の説明 |
| ショートカットキー | ページ表示中のみ有効な起動キー（連打回数条件可） |
| ホットキー | **非表示でも効く**グローバル起動キー（連打回数条件可、重複登録は旧割当解除） |
| マウス直接起動 | 非表示でも効くクリック/エッジ/サークル操作でのアイテム直接起動 |
| 同時起動アイテム | 1ボタンで複数アイテム一括起動。行ごとに起動遅延(ms)、負値でメインより先。連鎖禁止 |
| アイコン変更 | .exe/.dll/.ico/.icl から選択 + **オーバーレイアイコン**重ね表示。キャッシュ固定オプション |
| シェルメニュー | ONで右クリック=シェルコンテキストメニュー、Shift+右=アイテムメニュー |
| 多重起動制御 | 既に起動中なら: デフォルト（同じプログラムをもう一度起動）/アクティブ化/閉じる/何もしない。ウィンドウタイトル部分一致・クラス前方一致で対象絞り込み、対象プログラム変更、非表示ウィンドウ可視化、強制終了、「ウィンドウ参照」で実行中アプリから取得 |
| フック自動解除 | 起動前にフック解除→対象プロセス終了で自動再開（ゲーム対策） |
| フォルダー表示方法 | 開く / エクスプローラー / メニュー展開 |
| マウスオーバー自動展開 | サブメニュー/展開フォルダーをカーソルを載せるだけで展開 |

## 3. アイテムの種類と登録

- **登録手段**: D&D（複数一括可、Shift+ドロップでlnk変換なし）、アイテム登録ダイアログ
  （ファイル/URL入力、特殊フォルダー/特殊アイテムのプルダウン、**ウィンドウ参照**ボタン=
  実行中アプリのウィンドウをクリックしてパス自動取得。UWP/Office等の登録に使う）。
- ブラウザのアドレスバー/リンクのドロップで URL 登録。マイコンピュータ・ごみ箱・
  コントロールパネル内アイテムもドロップ可。
- **特殊フォルダー**: デスクトップ/コンピューター/ネットワーク/コントロールパネル/ごみ箱/
  ドキュメント/お気に入り/スタートメニュー/プログラム/スタートアップ他。
- **特殊アイテム**:
  - CLaunch制御系: ウィンドウロック/制限モード/フック解除/オプション/CLaunch終了
  - 追加機能系: 最近使用したアイテム（履歴）/最後に表示したメニュー/クリップボードフォルダー展開/
    スクリーンセーバー起動・停止/ディスプレイ電源OFF/Windows終了メニュー
  - 電源制御系: シャットダウン/再起動/サインアウト/スリープ/休止（確認メッセージ、1秒長押しで強制）
  - シェル管理系: タスクマネージャ/デバイスマネージャ/ディスクの管理
- **ごみ箱ボタン**: D&Dでごみ箱へ移動（Shiftで完全削除）、1秒長押しで空にする。

## 4. 起動まわりの挙動

- クリックまたは選択+Enter で起動。ドキュメント=関連付け、URL=既定ブラウザ。
- **連続起動モード**: Ctrl+左クリック or Space 起動。起動後もウィンドウが最前面に残り
  連続起動できる。非表示化 or 通常起動で自動解除。
- **登録ボタンへのファイルドロップ**: プログラム=引数として起動、ドキュメント=両方開く、
  フォルダー=コピー（Shiftで移動、デフォルト動作は設定可）。右ボタンドロップで
  選択メニュー（更新コピー/ショートカット作成含む）。
- **OLEドラッグ（逆方向D&D）**: 登録ボタンを右ボタンドラッグで外部アプリ/他ボタンへドロップ
  （「関連付け以外のプログラムで開く」等）。
- 履歴（最近使用したアイテム）: 最大件数・カテゴリ除外・個別除外・クリアを管理。

## 5. ページ・ボタン配置

- ページ（タブ）は最大100。タブD&Dで並び替え、タブのダブルクリックでページ設定。
- **行列座標で配置記憶**（4.20〜）: ウィンドウ幅を変えても配置不変。横ボタン数を超える登録は
  折り返さず**表示範囲外**へ→横スクロールで表示。
- 空白ボタン挿入（Insert）とボタン削除（Delete、空白だけでなく登録アイテムの削除にも使う）、
  空白削除・空行/空列の挿入/削除（空白ボタンメニューから）。シフト方向は「アイテムの登録方向」
  （横/縦）設定に従う。削除系操作でアイテム入りボタンは動かない。
- **ページ内スクロール**: 右ボタン+ホイール=縦、ホイール押し込み+回転=横。ページごとに
  スクロール自動リセットのタイミングを設定可（低頻度アイテムを下に隠す運用）。
- **ボタンロック**: Ctrl+右クリック。ロックボタンはページ切替・スクロールでも同位置に固定表示
  され全ページ共通のドックになる。スキンにロック中専用画像・ロックマーカーあり。
- **表示モード1/2**: 2つの表示モードをタイトルバーダブルクリック/Tabキーで切替。モード別に
  ウィンドウサイズ・スクロール状態・（オプションで）デザインとボタン配置を記憶。
  典型運用: モード1=正方形ボタンの厳選パネル、モード2=横長リストの全部入り。
  「アイテム起動後にモード1へ切替」オプションあり。

## 6. 検索

- Ctrl+F で検索ウィンドウ。即時絞り込み、**名前の先頭に近いマッチほど上位**。
- Tab でメインウィンドウとフォーカス交互切替。結果のプロパティを開くと当該ページへ切替。
- オプション: メインウィンドウへの**文字キー入力でそのまま検索開始**（ワード全削除で自動クローズ）、
  ツールチップ文字列も検索対象、起動後に検索ウィンドウを閉じる。

## 7. オプションダイアログ（16タブ）

クリック / エッジ / サークル / キーボード / マウス / フォルダー / メニュー / イベント /
無効化 / プラグイン / 表示 / スキン / デザイン1 / デザイン2 / ページ / その他。

要点:
- **キーボード・マウスタブ**: ほぼ全操作を「対象（ボタン/タイトルバー/メニュー等）×操作×機能」で再割り当て可能。
- **イベントタブ**: CLaunch起動時/終了時/省電力移行時/復帰時にアイテム自動起動（遅延付き）。
- **無効化タブ**: フルスクリーンウィンドウアクティブ時の自動無効化、アプリ単位の無効化
  （ウィンドウタイトル部分一致/クラス前方一致/実行ファイルパス）。制限モード（機能の一時停止、
  トレイアイコン変化）とフック解除（マウス/キーボードフック自体を外すゲーム対策）。
- **表示タブ**: 表示操作カテゴリ別の初期位置、常に手前に表示（ロック中のみ、のオプション）、
  待機表示、ツールチップ、トレイアイコン変更。
- **デザイン1/2タブ**: タイトルバー高さ（0で非表示）、枠幅、背景グラデーション、透明度
  （非アクティブ時のみ可）、キャプション、タブ・ボタンサイズ、アイコン取得サイズと表示サイズの分離、
  アイコン位置（上/左）、テキスト有無、フォント、登録方向（横/縦）。
- **ページタブ**: ページ別のデザイン（色/スキンパターン）→ タブの目印に使える。
- **その他タブ**: 定期設定バックアップ（INI 1ファイル、間隔日数）、相対パス登録（USB運用）、
  登録後にプロパティ表示、削除後に空白を詰める、ページ切替ループ、タブのマウスオーバーでページ切替。

## 8. スキンシステム

- 配置: Zip のまま `Skins/` へ、またはサブフォルダーに展開。中身は **skin.xml（UTF-16 LE XML）+
  パーツ画像（BMP/JPEG/PNG 対応。添付スキンは PNG）**。
- 問題があると **Classicスキン**（画像を使わない内蔵デザイン）へフォールバック。
- skin.xml 構造: `<skin title author version>` / `<general>`（tabPosition, alphaChannel
  ※alpha=0ピクセルはクリック不可, clippingColor, sizeArea, shadowArea）/ `<background>`
  （9スライス border, stretch等の method）/ `<frame>` / システムボタン `<closebutton>`
  `<pinbutton>` `<modebutton>` `<searchbutton>`（各 normal/hover/down/downhover）/
  `<caption>` / `<tabarea>` `<tab>`（active/inactive）/ `<buttonarea>` `<button>`
  （状態別+複数デザインパターン）/ `<lockedbutton>` `<lockmarker>` / `<submenu>`
  （フォルダー展開メニューと共有）/ `<searchwindow>` / `<recommended>`（mode1/mode2の推奨設定）。
- Classic スキン時はタブ/ボタンを色（グラデーション2色/単色、テキスト影/縁取り）で設定。
- 添付スキン: Glass / Solid Black / Vista-style。有志スキン配布サイトが公式からリンクされている。

## 10. その他の仕組み

- **サブメニュー**: ページ内をさらにグループ分けするメニュー型コンテナ。**入れ子可**。
  D&Dで出し入れ（ドラッグ中にボタンに載せると開いて挿入位置ライン表示）。複数選択
  （Ctrl/Shift）で一括起動・一括移動。左ドラッグ=移動、右ドラッグ=OLEドラッグ。
- **フォルダーのメニュー展開**: 階層ポップアップでファイラー的操作（Ctrl+X/C/V、F2リネーム、
  ソート順変更、新規フォルダー、パス名コピー Ctrl+Shift+C）。フィルタ・ソート・列数を
  アイテム単位に制御。ファイルドラッグ中はボタン上で自動展開されメニュー内へドロップ可。
- **ClAdmin**: UAC 環境で「管理者アプリがアクティブだと呼び出せない」「管理者起動すると
  標準権限エクスプローラーから D&D 不可」の両問題を解決する起動モジュール。
- **プラグイン**: `Plugins/` の DLL。公式配布: ウィンドウ操作/スキンチェンジャー。
  開発キット: github.com/pyonky/CLaunch-Plugins。言語モジュールで7言語対応。
- 設定バックアップ/復元は INI ファイルベース。相対パス⇔絶対パス一括変換。
