# CLaunch vs RLaunch現行版 ギャップ分析

「CLaunchを模倣して再作成」のロードマップ元ネタ。iced版の機能計画を立てるときに読む。
実装状況の正確な現行仕様は `rlaunch-legacy` スキルを参照。

## A. RLaunchに既にあるCLaunch相当機能（パリティ維持）

- トレイ常駐 / グローバルホットキートグル / デスクトップダブルクリック表示（WH_MOUSE_LL）
- フォーカス喪失自動非表示 + ピン留め（≒ウィンドウロック）
- ページ（タブ）+ ボタングリッド / タブD&D並び替え / ホイールでタブ切替
- D&D登録（複数一括、lnk解決）/ ファイル・フォルダ選択登録 / URL登録
- アイテムプロパティ（ラベル/引数/作業Dir/管理者/個別ホットキー/アイコン変更）
- 管理者起動 / ファイルの場所を開く / 検索（Ctrl+F）/ キーボードナビゲーション
- 多重起動防止 / マルチモニター（カーソルのあるモニターに表示）

## B. CLaunchにあってRLaunchに無い機能（模倣候補、実装難度: ★easy〜★★★hard）

### 呼び出し・非表示
| 機能 | 内容 | 難度 |
|---|---|---|
| カーソルアウトで非表示 | 現行はフォーカス喪失のみ。本家は「カーソルがウィンドウ外に出たら消える」が核 | ★★ |
| エッジファンクション | 画面端8箇所への移動で表示（認識範囲・遅延調整） | ★★（低レベルフック流用） |
| クリックファンクション拡張 | 任意ボタン×クリック種別×有効範囲の割り当て（現行はデスクトップダブルクリックのみ） | ★★★ |
| サークルファンクション | マウスで円を描いて表示 | ★★★ |
| ホットキー連打条件 | 「Ctrl 2回押し」等の連打回数条件 | ★★ |
| 待機表示 | 常時表示+使用時に移動、終わると待機位置へ戻る | ★★ |
| 表示操作別の初期ページ・表示位置 | 呼び出し方法ごとに開くページと位置を変える | ★★ |
| コマンドラインオプション | /n /l /m /s /p /a 等 | ★ |

### 起動・アイテム
| 機能 | 内容 | 難度 |
|---|---|---|
| 連続起動モード | Ctrl+クリック/Spaceで起動後もウィンドウ維持 | ★ |
| 同時起動アイテム | 1ボタンで複数起動、遅延指定 | ★★ |
| ボタンへのドロップ起動 | プログラムボタンにファイルをドロップ→引数起動、フォルダーへコピー/移動 | ★★ |
| OLE逆D&D | ボタンから外部アプリへドラッグ | ★★★ |
| 多重起動制御 | 起動中ならアクティブ化/閉じる等（ウィンドウ列挙・クラス/タイトルマッチ） | ★★ |
| 実行時の大きさ・優先度 | 最小化/最大化起動、プロセス優先度（現行は windowState 保存のみで未反映） | ★ |
| `<clipboard>` タグ | 引数にクリップボード文字列を埋め込み | ★ |
| ウィンドウ参照登録 | 実行中アプリのウィンドウクリックでパス取得（UWP対応） | ★★★ |
| 特殊アイテム | 電源制御/ごみ箱/スクリーンセーバー/タスクマネージャ等 | ★★ |
| 特殊フォルダー | shell: 名前空間（コントロールパネル等） | ★★ |
| 履歴（最近使用したアイテム） | 起動履歴メニュー+直接起動モード | ★★ |
| ショートカットキー（ページ内） | 表示中のみ有効なアイテム起動キー、頭文字自動割り当て | ★ |
| マウス直接起動 | 非表示状態からエッジ/クリック操作でアイテム直接起動 | ★★★ |

### 配置・表示
| 機能 | 内容 | 難度 |
|---|---|---|
| ページ内スクロール | グリッドをスクロールして表示範囲外アイテムに到達（現行は固定スロットのみ） | ★★ |
| 空白/空行/空列の挿入・削除 | Insert/Delete での配置編集 | ★ |
| ボタンロック | 全ページ共通位置に固定されるドック的ボタン | ★★ |
| 表示モード1/2 | 2モードの外観・配置・サイズを独立記憶して切替 | ★★ |
| フォルダーのメニュー展開 | 階層ポップアップ+ファイラー的操作+フィルタ/ソート | ★★★ |
| サブメニューの入れ子 | 現行グループはネスト不可 | ★★ |
| マウスオーバー自動展開 | グループ/フォルダーにカーソルを載せると開く（ドラッグ中は常時） | ★★ |
| オーバーレイアイコン | アイコンに重ねる識別マーク | ★ |
| タブ複数行・マウスオーバー切替・ページ別デザイン | タブバーの拡張 | ★★ |

### システム・運用
| 機能 | 内容 | 難度 |
|---|---|---|

| フルスクリーン時自動無効化 / アプリ別無効化 | 前面ウィンドウ監視 | ★★ |

| ClAdmin相当 | UAC 昇格問題の解決 | ★★★ |

## C. RLaunch独自機能（CLaunchに無い、維持するもの）

- **ウィジェット**（時計/CPUモニター等26種、Canvas描画）— 再実装方式は要設計判断
- カラーテーマJSON（25種=ビルトイン6+サンプル19、+外部追加、透過/Mica/Acrylic）
- グループのグリッド型ポップアップ（本家はメニュー型サブメニュー）
- タブ/グループ単位の個別リスト表示（本家は表示モード2で全体をリスト形式化できるのみ）
- 起動統計（回数・最終起動）、パス有効性チェック（⚠表示）
- エクスポート/インポート（JSON、マージ対応）
- フォルダブラウザウィンドウ、アイコンライブラリ（SVG46種）

## 優先度検討時の注意

- ユーザーは「現状の機能 + CLaunch模倣」を要望。まず A+C のパリティを確保し、
  B は「CLaunchらしさの核」（カーソルアウト非表示、エッジ、連続起動、ボタンロック、
  ドロップ起動）から優先するのが妥当。着手順はユーザーと合意すること。
