# ウィジェットの作り方ガイド

MyLauncher のウィジェットは、Canvas 2D API を使った小さな JavaScript プラグインです。  
`manifest.json`（設定定義）と `widget.js`（描画スクリプト）の 2 ファイルだけで作成できます。

---

## クイックスタート

### 1. テンプレートをコピー

ウィジェットフォルダにテンプレートが用意されています。

```
%APPDATA%/com.mylauncher.app/widgets/_template/
```

このフォルダをコピーしてリネームします（例: `my-widget`）。

> 設定画面のテーマ欄にある **📁** ボタン → 1つ上のフォルダ → `widgets/` でアクセスできます。

### 2. manifest.json を編集

ウィジェットのメタ情報と設定項目を定義します。

### 3. widget.js を編集

`draw()` 関数に Canvas 描画コードを記述します。

### 4. アプリを再起動

ウィジェット選択画面に新しいウィジェットが表示されます。

---

## フォルダ構成

```
widgets/
└── my-widget/          ← フォルダ名 = ウィジェット ID
    ├── manifest.json   ← メタ情報・設定スキーマ
    └── widget.js       ← 描画スクリプト
```

## manifest.json の書き方

```json
{
  "id": "my-widget",
  "label": "マイウィジェット",
  "author": "あなたの名前",
  "description": "ウィジェットの説明文",
  "emoji": "🧩",
  "version": "1.0.0",
  "updateInterval": 1000,
  "needsSystemInfo": false,
  "configSchema": [
    {
      "key": "textColor",
      "type": "color",
      "label": "文字色",
      "default": "#ffffff"
    },
    {
      "key": "backgroundColor",
      "type": "color",
      "label": "背景色",
      "default": "transparent"
    },
    {
      "key": "message",
      "type": "text",
      "label": "メッセージ",
      "default": "Hello!"
    }
  ]
}
```

### フィールド一覧

| フィールド | 型 | 必須 | 説明 |
|---|---|---|---|
| `id` | string | ○ | ウィジェットの一意 ID（フォルダ名と一致させる） |
| `label` | string | ○ | ウィジェット選択画面に表示される名前 |
| `author` | string | | 作者名（初期値: `"community"`） |
| `description` | string | | 説明文 |
| `emoji` | string | | 選択画面のアイコン絵文字（初期値: `"🧩"`） |
| `version` | string | | バージョン（初期値: `"1.0.0"`） |
| `updateInterval` | number | | 描画更新間隔（ミリ秒、初期値: `1000`） |
| `needsSystemInfo` | boolean | | `true` にすると CPU/メモリ使用率が `data.systemInfo` で渡される |
| `configSchema` | array | | 設定項目の定義（後述） |

## configSchema — 設定項目の定義

`configSchema` で定義した項目はウィジェット設定画面に自動でフォームが生成されます。

### フィールドタイプ

| type | UI | 追加プロパティ | 説明 |
|---|---|---|---|
| `color` | カラーピッカー | — | 色の選択 |
| `checkbox` | チェックボックス | — | ON / OFF |
| `select` | ドロップダウン | `options` | 選択肢から選ぶ |
| `text` | テキスト入力 | — | 自由テキスト |
| `number` | 数値入力 | `min`, `max`, `step` | 数値 |
| `datetime` | 日時ピッカー | — | 日時選択 |
| `file` | ファイル選択 | — | ファイルパス |

```json

      { "value": "Asia/Tokyo", "label": "東京" },
```

## widget.js の書き方

### 基本構造

widget.js は `draw` という名前の関数を定義します。  
この関数がフレームごとに呼び出されて描画を行います。

```javascript
function draw(ctx, w, h, config, data) {
  // 1. キャンバスをクリア
  ctx.clearRect(0, 0, w, h);

  // 2. 背景を描画（任意）
  if (config.backgroundColor && config.backgroundColor !== 'transparent') {
    ctx.fillStyle = config.backgroundColor;
    ctx.fillRect(0, 0, w, h);
  }

  // 3. メイン描画
  ctx.fillStyle = config.textColor || '#ffffff';
  ctx.font = Math.max(8, w * 0.15) + 'px sans-serif';
  ctx.textAlign = 'center';
  ctx.textBaseline = 'middle';
  ctx.fillText(config.message || 'Hello!', w / 2, h / 2);
}
```

### draw() の引数

| 引数 | 型 | 説明 |
|---|---|---|
| `ctx` | `CanvasRenderingContext2D` | Canvas 2D 描画コンテキスト |
| `w` | `number` | キャンバスの幅（ピクセル） |
| `h` | `number` | キャンバスの高さ（ピクセル） |
| `config` | `object` | `configSchema` で定義した設定値のオブジェクト |
| `data` | `object` | ランタイムデータ（後述） |

### data オブジェクト

| プロパティ | 型 | 説明 |
|---|---|---|
| `data.now` | `Date` | 現在時刻の Date オブジェクト |
| `data.systemInfo` | `object` | `{ cpu_usage: number, memory_usage: number }` — `needsSystemInfo: true` の場合のみ |
| `data.clicked` | `boolean` | ユーザーがウィジェットをクリックしたフレームで `true` |
| `data.invoke` | `function` | Rust コマンドを呼び出す関数（上級者向け） |

## 実用的なサンプル

### 例 1: ワールドクロック

世界各地の時刻を表示するシンプルなウィジェットです。

**manifest.json:**
```json
{
  "id": "world-clock",
  "label": "ワールドクロック",
  "emoji": "🌍",
  "updateInterval": 1000,
  "configSchema": [
    {
      "key": "timezone",
      "type": "select",
      "label": "都市",
      "default": "America/New_York",
      "options": [
        { "value": "America/New_York", "label": "New York" },
        { "value": "Europe/London", "label": "London" },
        { "value": "Asia/Tokyo", "label": "Tokyo" }
      ]
    },
    { "key": "textColor", "type": "color", "label": "文字色", "default": "#cdd6f4" },
    { "key": "backgroundColor", "type": "color", "label": "背景色", "default": "transparent" }
  ]
}
```

**widget.js:**
```javascript
function draw(ctx, w, h, config, data) {
  ctx.clearRect(0, 0, w, h);
  if (config.backgroundColor && config.backgroundColor !== 'transparent') {
    ctx.fillStyle = config.backgroundColor;
    ctx.fillRect(0, 0, w, h);
  }
  var tz = config.timezone || 'America/New_York';
  var timeStr;
  try {
    timeStr = data.now.toLocaleTimeString('ja-JP', {
      timeZone: tz, hour: '2-digit', minute: '2-digit', second: '2-digit'
    });
  } catch (e) {
    timeStr = data.now.toLocaleTimeString();
  }
  ctx.fillStyle = config.textColor || '#cdd6f4';
  ctx.textAlign = 'center';
  ctx.textBaseline = 'middle';
  ctx.font = 'bold ' + Math.max(10, w * 0.2) + 'px monospace';
  ctx.fillText(timeStr, w / 2, h / 2);
}
```

### 例 2: クリック対応ストップウォッチ

クリックでスタート／ストップするウィジェットです。  
グローバル変数でフレーム間の状態を保持します。

**widget.js:**
```javascript
// フレーム間で状態を保持するグローバル変数
var _sw = _sw || { running: false, elapsed: 0, lastTick: 0 };

function draw(ctx, w, h, config, data) {
  ctx.clearRect(0, 0, w, h);

  // クリックでスタート／ストップ切り替え
  if (data && data.clicked) {
    if (_sw.running) {
      _sw.running = false;
      _sw.lastTick = 0;
    } else {
      _sw.running = true;
      _sw.lastTick = Date.now();
    }
  }

  // 経過時間を計算
  var now = Date.now();
  if (_sw.running) {
    if (_sw.lastTick) _sw.elapsed += now - _sw.lastTick;
    _sw.lastTick = now;
  }

  var total = Math.floor(_sw.elapsed / 1000);
  var s = total % 60;
  var m = Math.floor(total / 60);
  var text = (m < 10 ? '0' : '') + m + ':' + (s < 10 ? '0' : '') + s;

  // 描画
  ctx.fillStyle = config.textColor || '#a6e3a1';
  ctx.font = 'bold ' + Math.max(10, w * 0.2) + 'px monospace';
  ctx.textAlign = 'center';
  ctx.textBaseline = 'middle';
  ctx.fillText(text, w / 2, h / 2);

  // ステータス
  ctx.font = Math.max(6, w * 0.08) + 'px sans-serif';
  ctx.fillStyle = _sw.running ? '#a6e3a1' : '#6c7086';
  ctx.fillText(_sw.running ? '● RUN' : '■ STOP', w / 2, h * 0.82);
}
```

## 高度な機能

### ウィジェットスパン（複数セル）

ウィジェット設定画面で **横スパン** と **縦スパン**（各 1〜4）を変更することで、複数セルにまたがる大きなウィジェットにできます。  
描画コード側では `w` と `h` の値が大きくなるので、自動的に拡大されます。

### システム情報の利用

`needsSystemInfo: true` を manifest に設定すると、`data.systemInfo` で CPU/メモリ使用率が渡されます。

```javascript
function draw(ctx, w, h, config, data) {
  ctx.clearRect(0, 0, w, h);
  var cpu = data.systemInfo ? data.systemInfo.cpu_usage : 0;
  ctx.fillStyle = '#ffffff';
  ctx.font = Math.max(10, w * 0.15) + 'px sans-serif';
  ctx.textAlign = 'center';
  ctx.textBaseline = 'middle';
  ctx.fillText('CPU: ' + Math.round(cpu) + '%', w / 2, h / 2);
}
```

### Rust コマンドの呼び出し

`data.invoke` を使って Rust バックエンドのコマンドを呼び出せます（上級者向け）。

```javascript
// 例: サウンドファイルの読み込み
if (data.invoke) {
  data.invoke('read_sound_file', { path: config.soundFile })
    .then(function(dataUrl) { /* ... */ });
}
```

## 描画のTips

### Canvas API の基本

```javascript
// テキスト描画
ctx.fillStyle = '#ffffff';
ctx.font = '14px sans-serif';
ctx.textAlign = 'center';       // 'left', 'center', 'right'
ctx.textBaseline = 'middle';    // 'top', 'middle', 'bottom'
ctx.fillText('テキスト', x, y);

// 矩形
ctx.fillStyle = '#313244';
ctx.fillRect(x, y, width, height);

// 円・弧
ctx.beginPath();
ctx.arc(cx, cy, radius, startAngle, endAngle);
ctx.strokeStyle = '#89b4fa';
ctx.lineWidth = 3;
ctx.stroke();

// 線
ctx.beginPath();
ctx.moveTo(x1, y1);
ctx.lineTo(x2, y2);
ctx.stroke();
```

### サイズへの適応

ウィジェットのサイズはユーザー設定やスパンによって変わります。  
フォントサイズやレイアウトは `w` と `h` を基準にした相対値にしてください。

```javascript
// ○ 推奨: w / h に基づく動的サイズ
ctx.font = Math.max(8, w * 0.15) + 'px sans-serif';

// × 非推奨: 固定サイズ
ctx.font = '14px sans-serif';
```

### 最初に clearRect

`draw()` 関数の先頭で必ず `ctx.clearRect(0, 0, w, h)` を呼んでください。  
前フレームの描画が残ると表示が崩れます。

### DPR は不要

デバイスピクセル比（DPR）のスケーリングは呼び出し元で処理済みです。  
`w` と `h` はそのまま論理ピクセルとして使用してください。
