# テーマの作り方ガイド

MyLauncher では CSS カスタムプロパティ（CSS 変数）ベースのテーマシステムを採用しています。  
JSON ファイルを 1 つ作成するだけで、独自の配色テーマを追加できます。

---

## クイックスタート

### 1. テーマフォルダを開く

設定画面のテーマ欄にある **📁** ボタンをクリックすると、テーマフォルダが開きます。

```
%APPDATA%/com.mylauncher.app/themes/
```

### 2. JSON ファイルを作成

テーマフォルダに `<テーマID>.json` ファイルを作成します。  
既存の JSON ファイルをコピーして編集するのが簡単です。

### 3. アプリを再起動

ファイルを保存してアプリを再起動すると、設定画面のテーマ一覧に表示されます。

---

## テーマ JSON の構造

```json
{
  "id": "my-theme",
  "label": "マイテーマ",
  "author": "あなたの名前",
  "variables": {
    "--bg-primary": "#1e1e2e",
    "--bg-secondary": "#181825",
    "--bg-button": "#313244",
    "--bg-button-hover": "#45475a",
    "--bg-button-active": "#585b70",
    "--bg-button-empty": "rgba(69, 71, 90, 0.25)",
    "--text-primary": "#cdd6f4",
    "--text-secondary": "#a6adc8",
    "--text-muted": "#6c7086",
    "--border-color": "#45475a",
    "--accent-color": "#89b4fa",
    "--accent-hover": "#74c7ec",
    "--shadow-color": "rgba(0, 0, 0, 0.4)",
    "--danger-color": "#f38ba8",
    "--success-color": "#a6e3a1",
    "--warning-color": "#fab387",
    "--border-radius": "8px",
    "--border-radius-sm": "4px"
  }
}
```

---

## 各フィールドの説明

### メタ情報

| フィールド | 必須 | 説明 |
|---|---|---|
| `id` | ○ | テーマの一意な識別子。ファイル名（拡張子なし）と一致させてください |
| `label` | ○ | 設定画面に表示されるテーマ名 |
| `author` | ○ | 作者名 |
| `variables` | ○ | CSS 変数の定義（キーと値のマップ） |

### CSS 変数リファレンス

#### 背景色

| 変数名 | 用途 |
|---|---|
| `--bg-primary` | メインウィンドウの背景 |
| `--bg-secondary` | タブバー・タイトルバー・ステータスバー |
| `--bg-button` | ボタン（通常状態） |
| `--bg-button-hover` | ボタン（ホバー時） |
| `--bg-button-active` | ボタン（押下時） |
| `--bg-button-empty` | 空きスロットの背景 |

#### テキスト色

| 変数名 | 用途 |
|---|---|
| `--text-primary` | 主要テキスト |
| `--text-secondary` | 補助テキスト |
| `--text-muted` | 薄いテキスト（ヒント等） |

#### UI 要素

| 変数名 | 用途 |
|---|---|
| `--border-color` | ボーダー・区切り線 |
| `--accent-color` | アクセントカラー（アクティブタブ等） |
| `--accent-hover` | アクセントカラーのホバー |
| `--shadow-color` | ボックスシャドウ |

#### ステータス色

| 変数名 | 用途 |
|---|---|
| `--danger-color` | 危険操作・エラー（削除ボタン等） |
| `--success-color` | 成功・正常（接続済み等） |
| `--warning-color` | 警告（パス無効マーク等） |

#### 角丸

| 変数名 | 用途 | 一般的な値 |
|---|---|---|
| `--border-radius` | ボタン・カード等の角丸 | `4px` 〜 `16px` |
| `--border-radius-sm` | 小さなUI要素の角丸 | `2px` 〜 `8px` |

## 透過テーマの作り方

背景が透けて見えるテーマを作成するには、以下の特殊変数を追加します。

### 追加の変数

| 変数名 | 値 | 説明 |
|---|---|---|
| `--window-opacity` | `0.0` 〜 `1.0` | ウィンドウ全体の不透明度 |
| `--window-effect` | `none` / `mica` / `acrylic` | Windows の透過エフェクト |

### エフェクトの種類

| エフェクト | 説明 |
|---|---|
| `none` | 透過エフェクトなし（通常表示） |
| `mica` | Windows 11 の Mica エフェクト（壁紙の色合いを反映した半透明） |
| `acrylic` | ぼかし＋ノイズの Acrylic エフェクト（すりガラス風） |

### 透過テーマの例

```json
{
  "id": "my-glass",
  "label": "マイグラステーマ",
  "author": "あなたの名前",
  "variables": {
    "--bg-primary": "rgba(30, 30, 46, 0.5)",
    "--bg-secondary": "rgba(24, 24, 37, 0.6)",
    "--bg-button": "rgba(49, 50, 68, 0.4)",
    "--bg-button-hover": "rgba(69, 71, 90, 0.5)",
    "--bg-button-active": "rgba(88, 91, 112, 0.6)",
    "--bg-button-empty": "rgba(69, 71, 90, 0.1)",
    "--text-primary": "#cdd6f4",
    "--text-secondary": "#a6adc8",
    "--text-muted": "#6c7086",
    "--border-color": "rgba(69, 71, 90, 0.5)",
    "--accent-color": "#89b4fa",
    "--accent-hover": "#74c7ec",
    "--shadow-color": "rgba(0, 0, 0, 0.2)",
    "--danger-color": "#f38ba8",
    "--success-color": "#a6e3a1",
    "--warning-color": "#fab387",
    "--border-radius": "12px",
    "--border-radius-sm": "6px",
    "--window-opacity": "0.85",
    "--window-effect": "acrylic"
  }
}
```

**ポイント:**
- 背景色には `rgba()` を使い、アルファ値で透過度を調整します
- `--window-effect` と `--window-opacity` の組み合わせで全体の透過感を調整します
- テキスト色は透過しない不透明色にすると読みやすくなります

---

## 配色のコツ

### ダークテーマ

- 背景は `#1e1e2e` 〜 `#2d2d3f` 程度の暗い色
- テキストは `#cdd6f4` 〜 `#ffffff` の明るい色
- コントラスト比 4.5:1 以上を目安に

### ライトテーマ

- 背景は `#eff1f5` 〜 `#ffffff` 程度の明るい色
- テキストは `#4c4f69` 〜 `#333333` の暗い色
- ボーダーは背景より少し暗い色

### アクセシビリティ

- WCAG AA（4.5:1）以上のコントラスト比を推奨
- ハイコントラストテーマ（ビルトイン）を参考にしてください

---

## 既存テーマの参照

テーマフォルダには、ビルトイン・サンプルテーマの JSON も含まれています。  
これらをコピーして編集するのが最も確実です。

| ファイル名 | 内容 |
|---|---|
| `dark.json` | 標準ダークテーマ |
| `light.json` | 標準ライトテーマ |
| `glass.json` | 透過ダークテーマ（参考用） |
| `wireframe.json` | スケルトンテーマ（参考用） |
