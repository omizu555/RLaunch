## 1. はじめに

MyLauncher は、Windows 向けのボタン型デスクトップランチャーです。  
グリッド状に並んだボタンにアプリ・URL・フォルダ・ウィジェットなどを配置し、ホットキー一つで呼び出して素早くアクセスできます。

### 主な特長

- **ホットキーで即呼び出し** — `Ctrl+Space` でどこからでも表示／非表示
- **ドラッグ＆ドロップ登録** — エクスプローラーからファイルをドロップするだけ
- **タブで分類管理** — 用途別にタブを分けてアイテムを整理
- **グループ（サブフォルダ）** — 関連するアイテムをグループにまとめて管理
- **ウィジェット** — 時計・CPU モニターなどの情報表示パネルを配置
- **豊富なテーマ** — 21 種のテーマ（ビルトイン6種 + サンプル15種）＋カスタムテーマ対応（透過テーマ含む）
- **フォルダブラウザ** — フォルダの中身をランチャーから階層的にブラウズ

## 2. 画面構成

```
┌─────────────────────────────────────────┐
│ ☰   MyLauncher              📌  ─  ✕  │ ← タイトルバー
├─────────────────────────────────────────┤
│ メイン │ 開発 │ ゲーム │       ＋       │ ← タブバー
├─────────────────────────────────────────┤
│                                         │
│   [Chrome]  [VSCode]  [Steam]  [ 📂 ]  │
│   [Terminal] [🕐時計] [  ]     [  ]     │ ← グリッド
│   [  ]      [  ]      [  ]     [  ]     │
│                                         │
├─────────────────────────────────────────┤
│ メイン — 4 アイテム / 12 スロット        │ ← ステータスバー
└─────────────────────────────────────────┘
```

### タイトルバー

| ボタン | 動作 |
|---|---|
| ☰ | 設定ウィンドウを開く |
| 📌 | ピン留め（自動非表示を一時停止） |
| ─ | ウィンドウを非表示にする |
| ✕ | ウィンドウを非表示にする（アプリは終了しません） |

タイトルバーのテキスト部分をドラッグするとウィンドウを移動できます。  
タイトルテキストは設定で変更できます。

### タブバー

タブをクリックして切り替えます。右端の **＋** ボタンで新しいタブを追加できます。

### グリッド

アイテムを配置するメインエリアです。各セルにはアプリ・URL・フォルダ・ウィジェット・グループを登録できます。

### ステータスバー

現在のタブ名、登録アイテム数、スロット総数を表示します。

## 3. 基本操作

### ランチャーの表示／非表示

| 操作 | 動作 |
|---|---|
| `Ctrl+Space` | ランチャーの表示／非表示を切り替え |
| デスクトップをダブルクリック | ランチャーを表示（カーソル位置に表示） |
| 他のウィンドウをクリック | 自動で非表示になる（ピン留め中は除く） |
| システムトレイアイコンをクリック | 表示／非表示を切り替え |

※ ホットキーは設定画面で変更できます。

> **多重起動防止**: MyLauncher は同時に 1 つしか起動できません。既に起動中の場合、2 つ目を起動しようとしても既存のインスタンスがアクティブになります。

### システムトレイ

タスクバーの通知領域（システムトレイ）に MyLauncher のアイコンが常駐しています。

| 操作 | 動作 |
|---|---|
| 左クリック | ランチャーの表示／非表示を切り替え |
| 右クリック | トレイメニューを表示 |

#### トレイメニュー

| メニュー | 動作 |
|---|---|
| 表示/非表示 | ランチャーの表示／非表示を切り替え |
| 設定 | 設定ウィンドウを開く |
| 終了 | アプリケーションを完全に終了する |

> **アプリの終了**: MyLauncher を完全に終了するには、トレイアイコンを右クリックして「終了」を選択してください。タイトルバーの ✕ ボタンはウィンドウを非表示にするだけで、アプリは終了しません。

### ピン留め

タイトルバーの 📌 ボタンをクリックすると **ピン留め** 状態になります。  
ピン留め中は、フォーカスが外れてもランチャーが自動で隠れません。  
もう一度クリックすると解除されます。

### 自動非表示の抑制

以下の状況では、ピン留めしていなくても自動非表示が抑制されます：

- ファイルをドラッグ＆ドロップ中
- ウィンドウをドラッグ移動中
- 設定ウィンドウを表示中
- グループポップアップを表示中
- フォルダブラウザを表示中

### 通知トースト

操作の完了時に、画面下部に通知メッセージが 3 秒間表示されます。  
例: 「アイテムを登録しました」「設定を保存しました」「N 件のアイテムを登録しました」

### アプリの起動

登録済みのアプリボタンを **クリック** するだけで起動します。  
起動後、ランチャーは自動で非表示になります（設定で変更可）。

## 4. アイテムの登録

### 方法 1: ドラッグ＆ドロップ（推奨）

1. エクスプローラーでファイルやショートカットを選択
2. ランチャーの空きセルにドラッグ＆ドロップ
3. 自動でアイコンと名前が取得されて登録完了

> **対応ファイル**: `.exe`, `.lnk`（ショートカット）, フォルダ, ドキュメント全般  
> `.lnk` ファイルはリンク先が自動解決されます。

### 方法 2: ファイル選択ダイアログ

1. 空きセルを **右クリック**
2. 「📁 ファイルを選択して追加」を選択
3. ファイル選択ダイアログでファイルを選択

> **ダブルクリック**: 空きセルを **ダブルクリック** しても同じダイアログが開きます。

### 方法 3: フォルダ選択ダイアログ

1. 空きセルを **右クリック**
2. 「📂 フォルダを選択して追加」を選択
3. フォルダ選択ダイアログでフォルダを選択

### 方法 4: URL 登録

1. 空きセルを **右クリック**
2. 「🌐 URLを登録」を選択
3. URL を入力（例: `https://google.com`）

### 複数ファイルの一括登録

エクスプローラーから複数ファイルを選択してまとめてドロップすると、空きセルに順番に登録されます。  
登録完了時に「N 件のアイテムを登録しました」と通知が表示されます。

## 5. アイテムの操作

### 起動

| 操作 | 動作 |
|---|---|
| クリック | 通常起動 |
| 右クリック →「▶ 起動」 | 通常起動 |
| 右クリック →「🛡 管理者として起動」 | 管理者権限で起動 |

### 並び替え

ボタンを **ドラッグ** して別のセルに移動すると、2 つのアイテムが入れ替わります。  
ドラッグ中は元のセルが半透明になり、移動先がハイライト表示されます。

> **タブをまたいで移動**: ドラッグ中に移動先のタブ上にホバーすると、0.5 秒後にそのタブに自動で切り替わります。そのまま目的のセルにドロップしてください。

> **グループへのコピー**: グループのポップアップが開いている状態でアイテムをドラッグし、ウィンドウ外でリリースすると、そのアイテムがグループの空きセルにコピーされます。

### 登録解除

1. アイテムを **右クリック**
2. 「🗑 登録解除」を選択
3. 確認ダイアログで「OK」をクリック

### ファイルの場所を開く

右クリック →「📂 ファイルの場所を開く」で、エクスプローラーで親フォルダを表示します。

### ホバーツールチップ

アイテムボタンにマウスカーソルを乗せると、ツールチップに以下の情報が表示されます：

- パス
- 作業ディレクトリ（設定されている場合）
- 起動回数
- 最終起動日時

### パスの有効性チェック

登録済みパスのファイルが移動・削除されている場合、ボタンに **⚠** マークが表示されます。  
チェックはタブ切り替え時に自動で実行されます。

## 6. アイテムの編集

アイテムを右クリック →「✏ 編集」で編集ダイアログが開きます。

| フィールド | 説明 |
|---|---|
| ラベル | ボタンに表示する名前 |
| パス | 実行ファイルのパスまたは URL |
| 引数 | 起動時の引数（省略可） |
| 作業ディレクトリ | 起動時の作業フォルダ（省略可） |
| ウィンドウ状態 | 通常 ／ 最大化 ／ 最小化 |
| 管理者として起動 | ON にすると常に管理者権限で起動 |
| グローバルホットキー | ランチャーを開かずに直接起動するキー（例: `Ctrl+Alt+C`） |

### アイコンの変更

アイテム編集ダイアログでアイコン画像をクリックすると、アイコンライブラリが開きます。  
ライブラリからアイコンを選択するか、カスタム画像ファイル（PNG / ICO）を選択してアイコンを変更できます。

`Ctrl+Enter` で保存、`Escape` で閉じます。

### グローバルホットキー

個々のアイテムにホットキーを設定すると、**ランチャーを表示することなく** そのアプリを直接起動できます。  
例えば `Ctrl+Alt+C` を Chrome に設定すると、どの画面からでもそのキーで Chrome が起動します。

### 起動統計

編集ダイアログにはアイテムの起動回数と最終起動日時が表示されます。

## 7. タブ管理

### タブの追加

タブバー右端の **＋** ボタンをクリックします。

### タブ名の変更

- タブを **ダブルクリック** してインライン編集
- または右クリック →「⚙ タブ設定」でタブ名を変更

### タブの切り替え

| 操作 | 動作 |
|---|---|
| タブをクリック | そのタブに切り替え |
| タブバー上でマウスホイール | 前後のタブに切り替え |

### タブの並び替え

タブを **ドラッグ** して好きな位置に移動できます。

### タブの複製

右クリック →「📋 タブを複製」で、全アイテムをコピーした新しいタブが作成されます。

### タブ設定

右クリック →「⚙ タブ設定」で、タブの各種設定をまとめて変更できます。

| 設定項目 | 説明 |
|---|---|
| タブ名 | タブの表示名 |
| グリッドサイズ | 列数（1〜20）× 行数（1〜10） |
| 表示モード | グリッド / リスト（タブ個別 or 全体設定に従う） |
| リスト列数 | 1〜4 列（タブ個別 or 全体設定に従う） |

### タブの削除

右クリック →「🗑 タブを削除」で削除します。  
アイテムが登録されている場合は確認ダイアログが表示されます。  
最後の 1 タブは削除できません。削除されたタブの隣のタブが自動で選択されます。

## 8. グループ機能

グループを使うと、複数のアイテムを 1 つのボタンにまとめて管理できます。

### グループの作成

1. 空きセルを **右クリック**
2. 「📁 サブグループを作成」を選択
3. グループ名・アイコン・カラー・グリッドサイズを設定
4. 「作成」をクリック

### グループの設定

| 項目 | 内容 |
|---|---|
| グループ名 | 任意のテキスト |
| アイコン | 16 種のアイコンから選択（📂 📁 🗂 📦 🎮 🛠 📝 🎵 🎨 📊 🌐 ⭐ 🔧 💻 📸 🎬）またはアイコンライブラリ |
| アイコンカラー | 8 色から選択 ＋ デフォルト色 |
| グリッドサイズ | 2〜8 列 × 1〜6 行 |
| 表示モード | グリッド / リスト（親設定を継承 or 個別設定） |
| リスト列数 | 1〜4 列（親設定を継承 or 個別設定） |

グループの表示モード・リスト列数は、デフォルトでは親タブ（または全体設定）を継承します。  
グループごとに個別設定することもできます。

### グループの使い方

1. グループボタンを **クリック** するとポップアップウィンドウが開きます
2. ポップアップはカーソル位置を基準に表示されます（マルチモニター対応）
3. ポップアップ内でアイテムを起動・管理できます
4. ポップアップ外をクリックすると自動で閉じます
5. メインウィンドウがフォーカスを取得すると、ポップアップは自動で閉じます

> **ポップアップ閉じ後の動作**: ポップアップが閉じた後、メインウィンドウにフォーカスがなければメインウィンドウも自動で非表示になります。

### グループ内のアイテム登録

メイングリッドと同じ方法でアイテムを登録できます。

| 方法 | 操作 |
|---|---|
| ドラッグ＆ドロップ | エクスプローラーからファイルをポップアップにドロップ（複数ファイル対応） |
| メインからドラッグ | メイングリッドのアイテムをドラッグしてウィンドウ外にリリース → グループにコピー |
| ダブルクリック | 空きセルをダブルクリック → ファイル選択ダイアログ |
| ファイル選択 | 空きセル右クリック →「📁 ファイルを選択して追加」 |
| フォルダ選択 | 空きセル右クリック →「📂 フォルダを選択して追加」 |
| URL登録 | 空きセル右クリック →「🌐 URLを登録」 |
| ウィジェット配置 | 空きセル右クリック →「🕐 ウィジェットを配置」 |

### グループ内のアイテム操作

アイテムの右クリックメニューはメイングリッドと同等の機能を提供します。

| 操作 | 説明 |
|---|---|
| ▶ 起動 | アイテムを起動（クリックでも起動可能。起動後ポップアップは自動で閉じます） |
| 📂 フォルダを参照 | フォルダ型アイテム: フォルダブラウザを表示 |
| 🛡 管理者として起動 | 管理者権限で起動 |
| 🔄 クリック動作切替 | フォルダ型: 「開く ↔ 参照」を切替 |
| 📂 ファイルの場所を開く | エクスプローラーでファイル位置を表示 |
| ✏ 編集 | アイテムのプロパティ（ラベル、パス、アイコン等）を編集 |
| 📋 パス表示 | 登録パスを表示 |
| 🗑 登録解除 | アイテムをグループから解除（確認ダイアログ付き） |

**ウィジェットの右クリックメニュー:**

| 操作 | 説明 |
|---|---|
| ⚙ ウィジェット設定 | ウィジェットの設定を変更 |
| 🔄 ウィジェットを変更 | 別のウィジェットに置き換え |
| 🗑 ウィジェットを解除 | ウィジェットを除去 |

**その他の操作:**
- **並べ替え**: アイテムをドラッグして入れ替え

### グループの編集・削除

- 右クリック →「✏ グループ名を変更」でアイコンやサイズも含めて再編集
- 右クリック →「🗑 グループを削除」で削除

## 9. ウィジェット

グリッドに情報パネルを配置できます。  
時計・CPU モニター・カウントダウンタイマーなど多彩なウィジェットを利用できます。

### ウィジェットの配置

1. 空きセルを **右クリック**
2. 「🕐 ウィジェットを配置」を選択
3. ウィジェット選択ウィンドウで種類を選択

### ビルトインウィジェット（6 種）

| 名前 | 説明 |
|---|---|
| アナログ時計 | 針で時刻を表示 |
| デジタル時計 | 数字で時刻を表示 |
| カウントダウンタイマー | 指定時刻までのカウントダウン |
| CPU モニター | CPU 使用率をリアルタイム表示 |
| メモリモニター | メモリ使用量をリアルタイム表示 |
| 日付カレンダー | 今日の日付を表示 |

### プラグインウィジェット（20 種）

| 名前 | 説明 |
|---|---|
| ワールドクロック | 世界各地の時刻 |
| ポモドーロタイマー | 集中タイマー |
| ストップウォッチ | 経過時間計測 |
| サイコロ | ランダムなサイコロ |
| 呼吸ガイド | リラックス呼吸パターン |
| バッテリー残量 | PC バッテリーの状態 |
| カラーホイール | 回転するカラーパレット |
| バイナリ時計 | 2 進数で時刻表示 |
| 月齢 | 現在の月の満ち欠け |
| 年進捗 | 今年の経過パーセント |
| 名言 | 日替わり名言表示 |
| パーティクル | 粒子アニメーション |
| マトリックスレイン | Matrix 風エフェクト |
| ネットワーク状態 | ネットワーク接続状況 |
| パルスアニメーション | 脈動するアニメーション |
| クイックメモ | 簡易メモ帳 |
| 天気アイコン | 天気を表すアイコン |
| 日照情報 | 日の出・日の入り |
| FPS カウンター | フレームレート表示 |
| デュアルモニター | マルチモニター情報 |

### ウィジェットの設定

ウィジェットを右クリック →「⚙ ウィジェット設定」で、更新間隔やウィジェット固有の設定を変更できます。

### ウィジェットのリサイズ（スパン）

ウィジェット設定画面で **横スパン** と **縦スパン**（各 1〜4）を変更することで、複数セルにまたがる大きなウィジェットを配置できます。

### ウィジェットの変更・解除

| 操作 | 動作 |
|---|---|
| 右クリック →「🔄 ウィジェットを変更」 | 別のウィジェットに差し替え |
| 右クリック →「🗑 ウィジェットを解除」 | セルをクリア |

### プラグインウィジェットの追加

独自のウィジェットを追加できます。以下の構成で `AppData/widgets/` フォルダに配置します。

```
AppData/widgets/<widget-id>/
├── manifest.json   … メタ情報・設定スキーマ
└── widget.js       … 描画スクリプト（draw 関数）
```

詳しくは「[ウィジェットの作り方ガイド](ウィジェットの作り方.md)」を参照してください。

## 10. 検索機能

### 検索の開き方

`Ctrl+F` キーで検索バーを表示します。

### 使い方

1. テキストを入力すると即座に検索結果が表示されます（インクリメンタルサーチ）
2. 全タブのアイテムが対象で、ラベルとパスの両方で検索します
3. 結果には所属タブ名も表示されます

### 検索結果の操作

| 操作 | 動作 |
|---|---|
| `↑` / `↓` | 選択項目の移動 |
| `Enter` | 選択中のアイテムを起動 |
| `Ctrl+Enter` | 選択中のアイテムがあるタブに移動 |
| 結果をクリック | アイテムを起動 |
| `Escape` | 検索バーを閉じる |

## 11. フォルダブラウザ

登録済みフォルダの中身をランチャー上で階層的にブラウズできます。

### 開き方

- フォルダアイテム（ブラウズモード）をクリック
- フォルダアイテムを右クリック →「📂 フォルダを参照」

フォルダブラウザはカーソル位置を基準に表示されます（マルチモニター対応）。  
タイトルバーには現在表示中のフォルダ名が表示されます。

### 操作

| 操作 | 動作 |

| フォルダをクリック | サブフォルダに移動 |
| ファイルをクリック | ファイルを起動してウィンドウを閉じる |
| `⬆` ボタン / `Backspace` | 上のフォルダへ戻る |
| 「📂 Explorer で開く」 | 現在のフォルダをエクスプローラーで開く |
| `Escape` | ウィンドウを閉じる |

### フォルダのクリック動作の切り替え

フォルダアイテムは 2 つのクリック動作を切り替えられます：

| モード | 動作 | 切り替え方 |

## 12. 設定

タイトルバーの **☰** ボタン、またはシステムトレイの右クリックメニューから設定画面を開きます。

### 設定項目一覧

| 設定 | 初期値 | 説明 |
|---|---|---|
| テーマ | Dark | 配色テーマ（21 種＋カスタム） |
| ホットキー | `Ctrl+Space` | 表示／非表示キー |
| 表示位置 | カーソル位置 | `カーソル位置` / `画面中央` / `前回の位置を記憶` |
| 表示モード | グリッド | `グリッド (アイコン表示)` / `リスト (コンパクト表示)` |
| 列数 | 8 | グリッドの列数（4〜16） |
| 行数 | 4 | グリッドの行数（2〜10） |
| セルサイズ | 64px | 各セルの大きさ（40〜120px） |
| 自動起動 | OFF | Windows 起動時に自動実行 |
| 起動後に非表示 | ON | アプリ起動後にランチャーを自動で隠す |
| ラベルを表示 | ON | ボタン下のラベル表示 |
| ラベルフォントサイズ | 10px | ラベルの文字サイズ（8〜16px） |
| アプリタイトル | MyLauncher | タイトルバーに表示するテキスト（空欄で非表示） |

> グリッドの列数・行数を変更すると、すべてのタブに適用されます。既存アイテムは保持されます。

### セルサイズのクイック変更

`Ctrl+マウスホイール` でセルサイズを段階的に変更できます（設定画面を開かずに調整可能）。  
対応サイズ: 40 → 48 → 56 → 64 → 72 → 80 → 96 → 112 → 120 px

### 表示モード

設定画面の「表示モード」で、ランチャーの表示スタイルを切り替えられます。
この設定は全体のデフォルト値です。タブごとに個別の表示モードを設定することもできます。

| モード | 説明 |
|---|---|
| グリッド (アイコン表示) | 従来のアイコンボタンによるグリッド表示。正方形セルにアイコン + ラベルを配置 |
| リスト (コンパクト表示) | グリッドと同じ機能だがセルが横長の長方形になるコンパクト表示。アイコン + ラベルを横並びに配置 |

リストモードはグリッドと同じエンジンを使用しており、以下の機能がそのまま利用可能です:
- ウィジェットの表示と操作（セルサイズに合わせて自動リサイズ）
- グループの表示と操作
- ドラッグ＆ドロップによる並べ替え
- 右クリックコンテキストメニュー
- キーボードナビゲーション
- 空セルの表示

### リスト列数

リストモードでは列数を 1～4 の範囲で設定できます。
設定画面の「リスト列数」で全体のデフォルト値を設定し、タブごとに個別設定も可能です。

### タブ個別の表示設定

タブを右クリック →「⚙ タブ設定」で、表示モード・リスト列数をタブごとに個別設定できます。

- **全体設定を使用**: チェックすると全体設定のデフォルト値を使用
- **個別設定**: チェックを外すとタブ固有の設定を指定可能

タブ個別設定は全体設定より優先されます。

### ビルトインテーマ（6 種）

| テーマ | 説明 |

### サンプルテーマ（15 種）

サンプルテーマはアプリにリソースとしてバンドルされていますが、自動的にはインストールされません。
使用するには以下の手順でテーマフォルダにコピーしてください：

1. 設定画面のテーマ欄にある **📦** ボタンでサンプルテーマフォルダを開く
2. 使用したいテーマの JSON ファイルをコピー
3. **📁** ボタンでテーマフォルダを開き、コピーしたファイルを貼り付け
4. アプリを再起動すると新しいテーマが一覧に表示される

| テーマ | 説明 |
|---|---|
| Nord | 北欧の冬を思わせる冷たいブルー |
| Solarized Dark | 目に優しい計算されたカラーパレット |
| Monokai | コーディングの定番、ビビッドなアクセント |
| Dracula | 人気のダークテーマ、パープル系 |
| Gruvbox Dark | レトロでウォームな配色 |
| Tokyo Night | 深い夜空のようなテーマ |
| Rosé Pine | やわらかなピンクとパープルのエレガントなテーマ |
| Everforest | 自然のグリーンをベースにした森テーマ |
| Cyberpunk | ネオンカラーの近未来テーマ |
| Sunset Glow | 夕焼けの暖かいグラデーション |
| ハイコントラスト | WCAG AAA 準拠の高視認性テーマ |
| Glass Dark (透過) | 半透明ダーク＋ Acrylic エフェクト |
| Glass Light (透過) | 半透明ライト＋ Mica エフェクト |
| Wireframe (スケルトン) | 背景がほぼ透明、線とテキストのみ表示 |
| Frosted (すりガラス) | 紫系のすりガラス風透過テーマ |

### 透過テーマについて

「Glass」「Wireframe」「Frosted」は Windows の透過効果（Acrylic / Mica）を利用した透過テーマです。  
背景が透けて見え、デスクトップの壁紙やウィンドウが薄く見える効果があります。  
透過設定はテーマごとに自動適用されるため、設定画面での個別操作は不要です。

> **注意**: 透過効果は Windows 10/11 のみ対応しています。ハードウェア構成や Windows の設定によっては正しく表示されない場合があります。

### ハイコントラストモード

Windows のハイコントラスト設定が有効な場合、自動的にハイコントラスト配色に切り替わります。  
また、テーマ一覧に「High Contrast」テーマも用意されています。

### テーマのリアルタイムプレビュー

設定画面でテーマやセルサイズなどを変更すると、保存前にメインウィンドウに即座にプレビューが反映されます。  
設定を保存せずにキャンセルすると元のテーマに戻ります。

### 子ウィンドウへのテーマ自動同期

グループポップアップ・フォルダブラウザ・設定ウィンドウなどの子ウィンドウにも、メインと同じテーマが自動で適用されます。  
透過テーマの場合、子ウィンドウにも透過効果が反映されます。

### カスタムテーマの追加

1. 設定画面のテーマ欄にある **📁** ボタンでテーマフォルダを開く
2. 既存テーマの JSON をコピーして編集
3. アプリを再起動すると新しいテーマが一覧に表示される

詳しくは「[テーマの作り方ガイド](テーマの作り方.md)」を参照してください。

## 14. エクスポート／インポート

設定画面のフッター部分にボタンがあります。

| ボタン | 動作 |
|---|---|

> エクスポートデータには設定・タブ・アイテム・グループ・ウィジェットがすべて含まれます。  
> PC 移行時にはエクスポート → インポートで環境を再現できます。

## 20. トラブルシューティング

### ランチャーが表示されない

- タスクトレイにアイコンがあるか確認してください
- ホットキー（初期: `Ctrl+Space`）を押してみてください
- 他のアプリがホットキーを使用している場合、設定でホットキーを変更してください
- 既に別のインスタンスが起動している可能性があります（多重起動防止機能により、1 つしか起動できません）

### アプリが起動しない

- アイテムに **⚠** マークが表示されている場合、ファイルが移動・削除されています
- 右クリック →「✏ 編集」でパスを修正してください
- 管理者権限が必要なアプリは、右クリック →「🛡 管理者として起動」を試してください

### データを復元したい

1. 設定画面で「フォルダを開く」をクリック
2. `launcher-data.bak.1.json` を `launcher-data.json` にリネーム（元のファイルは別名で退避）
3. アプリを再起動

またはエクスポート済みの JSON ファイルがある場合、設定画面の「📥 インポート」からインポートしてください。

### テーマが反映されない

- カスタムテーマを追加した場合、アプリの再起動が必要です
- テーマフォルダ内の JSON が正しいか確認してください

### ウィンドウが見えない場所に表示される

- 設定の「表示位置」を「画面中央」に変更してください
- マルチモニター環境でモニターを外した場合に発生することがあります

### マルチモニターでの動作

- ホットキーやデスクトップダブルクリックでランチャーを表示すると、カーソルがあるモニターに表示されます
- グループポップアップもカーソルがあるモニターの作業領域内に表示されます
